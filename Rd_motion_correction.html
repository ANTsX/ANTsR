Motion-Correction¶

Motion Correction

description:

usage:

motion_correction( img )

examples:

moco_results <- motion_correction( “input_img.nii” )

# motion-corrected image

moco_results$moco_img

# motion-correction parameters

moco_results$moco_params

# average image

moco_results$moco_avg_img
