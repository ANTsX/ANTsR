=========================
temporalwhiten 
=========================

purpose:  

 Simple autocorrelation-based temporal whitening function.
 
description:  

 Temporally whitens the input matrix using autoregression and returns the result.
 

usage:  

 wmat<-temporalwhiten( mat  )
 
examples: 

.. sourcecode:: r
    

    mat <- matrix(c(rep(1, 100), rep(0, 200)), ncol = 50)
