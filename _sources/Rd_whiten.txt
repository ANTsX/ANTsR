=========================
whiten 
=========================

purpose:  

 Simple whitening function.
 
description:  

 Whitens the input matrix using SVD and returns the result.
 

usage:  

 wmat<-whiten( mat  )
 
examples: 

.. sourcecode:: r
    

    mat <- matrix(c(rep(1, 100), rep(0, 200)), ncol = 50)
